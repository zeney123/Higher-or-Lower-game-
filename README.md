# Higher or Lower game 
A simple python game where you guess which celebrity or brand has more Instagram followers. It was built using basic python concepts like loops, conditionals, random selection, and dictionaries.

This project was created to practice:

Working with lists & dictionaries

Game loops (while loops)

Comparing values

Using random.choice()

Printing formatted output

*How to Play*

You’ll be shown Two options: A and B, each with:

Name

Description

Country

Guess which one has more followers by typing:

A

or

B

If you’re right → your score increases, and the round continues.

If you’re wrong → the game ends and your final score is shown.
